- - 2008 年：厄瓜多尔基多的 Quito Ultratumba 夜总会火灾，造成 19 人死亡，至少 24 人受伤。
- - 2005年：红衣主教约瑟夫·拉辛格当选为教皇，成为教皇本笃十六世。

- - 1995年：俄克拉荷马城爆炸案：美国俄克拉荷马城的阿尔弗雷德·P·默拉联邦大楼遭到轰炸，造成168人死亡，其中包括19名六岁以下儿童。

- - 1985年：200名烟酒枪械管理局和联邦调查局特工围攻阿肯色州白人至上主义生存主义组织“圣约、圣剑和圣臂”的大院；两天后CSA投降。
- - 1984 年：Advance Australia Fair 被宣布为澳大利亚国歌，绿色和金色为国色。

- - 1973 年：葡萄牙社会党在德国巴特明斯特艾费尔镇成立。
